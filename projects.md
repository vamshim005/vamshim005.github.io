## Smart city Traveller

Smart city traveller is a website and an app, this helps users to find a perfect place for their needs and preferences. Generally, whenever we want to go to a particular place we will search in chrome, Mozilla, Internet explorer and based on rating we will choose and go to that place. In Smart City user can sign up or login and can find a perfect place for their needs and preferences based on comments of previous users which is performed using sentiment analysis i.e., the process of computationally identifying and categorizing opinions expressed in a piece of text, especially in order to determine whether the writer’s attitude towards a particular topic, product, etc., is positive, negative, or neutral . After successful login user can find top 4 places of their choice based on comments and using google map they can find the route to reach the destination.

 **Sentiment Analysis**  
   Predict sentiment from sentences by training a LSTM or GRU based sequential model on your own dataset. The models will be trained from scratch.

**Tools Used**: Android Studio, Java, JavaScript, AngularJS, Boot strap

## Voice based Biometrics

One of the growing technology in security systems today is biometrics. Biometric security system is used to authenticate and provide access based on instant verification of an individual’s physical characteristics. 
As it evaluates biological data, it is strongest and most foolproof physical security technique.
This is an android application which protects private data through voice biometrics. It provides a locking mechanism where in password will be user’s voice.

**Domain**: Android Application (Android studio), Machine Learning (musicG API)

## Sentiment Analysis using ELMo and BERT

9873 reviews of films from various customers were collected. To identify if the review was a positive/ negative/
neutral review and preprocessing were done along with text normalization.
Imported and used pretrained ELMO model from TensorFlow and google BERT model, where we extracted ELMO
vectors and BERT vectors for the cleaned tweets.
Used classification models like SVM and got an evaluation score of 89% with each one of these vectors.

## AWS Cloud Application - Image Recognition as a Service

Developed Web Application – IaaS model of cloud service that provides image classification as a service using
AWS cloud resources.
Designed a multi-tier architecture with a single web tier and auto-scaling app tier based on input requests traffic
in the SQS queue and used S3 Buckets to maintain persistency.
Implemented a load balancer and controller to ensure the elasticity of the application automatically by scaling
out and in on demand.

## Large-scale Geo-Spatial Data Analysis

Performed hot zone and hot cell analysis on large-scale spatial data using Spark to identify and rank the top 30
hot zones.
Performed spatial queries on large database containing geographic data and real time location data and helped a
taxi firm in New York in their operational and strategic decisions. Used Apache Spark, Scala, and AWS for this
project.
Analyzed New York’s Yellow cab taxi dataset to recognize high pick-up probability areas within New York and
increased the pick-up rate by 40% by abiding to recommendations from the analysis.
